def extract_frames(video_path, fps=30):
    cap = cv2.VideoCapture(video_path)
    frames = []
    while True:
        ret, frame = cap.read()
        if not ret:
            break
        frames.append(frame)
    return frames

📊 Gold Standard Metric Calculations
🎯 Wrist Speed
speed = distance(wrist_t, wrist_t-1) / delta_time
🎯 Elbow Angle
Using cosine rule between:
Shoulder → Elbow → Wrist

🎯 Shoulder Rotation Velocity
Track shoulder-to-hip angle over time.

🎯 Balance Stability Score
Measure center of mass horizontal drift:

stability = 100 - normalized(hip_variance)
🎯 Ball Speed
ball_speed = distance(ball_t, ball_t-1) / delta_time
🎯 Spin Estimation (Approximation)
Based on:

Arc curvature

Frame-to-frame deviation

🎯 Rhythm Consistency
Standard deviation of:

backswing duration

follow-through duration

Lower deviation = higher score.

def normalize(value, min_val, max_val):
